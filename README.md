# Crud Básico NodeJs + MySQl


## Instalação

Você deve possuir o Node instalado em sua máquina e em seguida entrar na pasta do projeto e digitar o seguinte comando:

```bash
npm install
```

## Modo de Usar

```bash
Após a instalação de todas as dependências, execute:

npm run dev
```
